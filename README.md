#  Study Time vs Marks Analysis

##  Project Overview
This project analyzes the relationship between students' study hours and their academic performance using Python. It aims to understand how study time influences marks through data visualization and basic analysis.

##  Problem Statement
The objective of this project is to examine whether increasing study hours leads to improved academic performance.

##  Solution
The project uses Python to analyze sample data representing study hours and corresponding marks. A scatter plot is generated along with a best-fit line to visualize the relationship and identify trends.

##  Tools and Technologies
- Python  
- NumPy  
- Matplotlib  

##  How to Run

1. Install Python on your system  
2. Install the required libraries:
   pip install numpy matplotlib  
3. Run the script:
   python analysis.py  

##  Output
- Scatter plot representing study hours versus marks  
- Best-fit line showing the trend  
- Predicted marks based on user input  

##  Key Learnings
- Data visualization techniques  
- Understanding linear relationships  
- Practical use of Python libraries
